# A 3D Software Renderer

A [Software Renderer](https://en.wikipedia.org/wiki/Software_rendering)
### Dependencies

- [SDL2](http://libsdl.org)
- gcc
- make

#### Debian

```bash
  sudo apt install libsdl2-2.0-0 libsdl2-dev
```

#### Fedora
```bash
  sudo dnf install SDL2 SDL2-devel
```
### Compiling

```bash
make
```
